# 希尔伯特曲线 vs 莫顿码 - 空间填充曲线对比工具

这是一个交互式网页应用,用于可视化和对比两种重要的空间填充曲线:**希尔伯特曲线(Hilbert Curve)** 和 **莫顿码(Morton Code/Z-order曲线)**。

## 🎯 功能特性

### 核心功能
- ✨ **实时可视化**: 同时展示希尔伯特曲线和莫顿码的生成过程
- 🎬 **动画演示**: 支持播放、暂停、重置动画,直观展示曲线的遍历顺序
- 🎚️ **可调参数**: 
  - 曲线阶数 (1-6阶)
  - 动画速度 (0.5x - 3.0x)
  - 显示选项 (网格、编号、动画模式)
- 📊 **实时统计**: 显示当前位置、坐标、进度百分比等信息
- 📋 **特性对比表**: 详细对比两种曲线的优缺点和应用场景

### 视觉设计
- 🌙 现代深色主题
- 🎨 渐变色和玻璃态效果
- ✨ 流畅的动画和过渡效果
- 📱 完全响应式设计,支持各种屏幕尺寸

## 🎮 操作指南

### 控制面板说明

1. **曲线阶数滑块**
   - 范围: 1-6
   - 阶数越高,曲线越复杂,单元格越多
   - 阶数n对应 2^(2n) 个单元格
   - 例: 阶数3 = 8×8 = 64个单元格

2. **动画控制按钮**
   - 🎬 **播放**: 开始动画演示
   - ⏸️ **暂停**: 暂停当前动画
   - 🔄 **重置**: 重置到初始状态

3. **显示选项**
   - ☑️ **显示网格**: 显示/隐藏背景网格
   - ☑️ **显示编号**: 显示/隐藏单元格序号
   - ☑️ **动画模式**: 启用/禁用动画效果

4. **动画速度滑块**
   - 范围: 0.5x - 3.0x
   - 调整动画播放速度

### 信息面板说明

每个曲线下方显示:
- **总单元数**: 当前阶数下的总单元格数量
- **当前位置**: 动画当前遍历到的索引
- **坐标**: 当前位置的(x, y)坐标
- **进度指示器**: 显示完成百分比

## 📚 算法说明

### 希尔伯特曲线 (Hilbert Curve)

**特点:**
- 连续的分形空间填充曲线
- 优秀的空间局部性保持特性
- 相邻空间位置在曲线上距离也近

**算法核心:**
- 递归分治思想
- 通过旋转和翻转四个象限
- 时间复杂度: O(n²)

**应用场景:**
- 图像处理和压缩
- 空间数据索引
- 地理信息系统(GIS)
- 数据库查询优化

### 莫顿码 (Morton Code / Z-order)

**特点:**
- 基于位交织(bit interleaving)
- 计算简单,效率极高
- Z字形遍历模式

**算法核心:**
- 将x和y坐标的二进制位交织
- 纯位运算,无需递归
- 时间复杂度: O(log n)

**应用场景:**
- 数据库多维索引
- 四叉树/八叉树线性化
- 点云数据组织
- 游戏引擎空间划分

## 🔬 特性对比

| 特性 | 希尔伯特曲线 | 莫顿码 |
|------|------------|--------|
| **空间局部性** | ⭐⭐⭐⭐⭐ 优秀 | ⭐⭐⭐⭐ 良好 |
| **计算复杂度** | ⭐⭐⭐ 中等 | ⭐⭐⭐⭐⭐ 简单 |
| **编码效率** | ⭐⭐⭐ 较慢 | ⭐⭐⭐⭐⭐ 快速 |
| **解码效率** | ⭐⭐⭐ 较慢 | ⭐⭐⭐⭐⭐ 快速 |
| **范围查询** | ⭐⭐⭐⭐⭐ 优秀 | ⭐⭐⭐⭐ 良好 |

### 选择建议

**选择希尔伯特曲线,如果:**
- 需要最佳的空间局部性
- 进行范围查询和聚类分析
- 图像处理和数据压缩
- 对计算效率要求不是特别高

**选择莫顿码,如果:**
- 需要快速的编解码
- 实现简单的空间索引
- 内存受限的环境
- 需要硬件加速(位运算)

### 主要类

1. **HilbertCurve**: 希尔伯特曲线算法实现
   - `indexToPoint()`: 索引转坐标
   - `rotate()`: 旋转/翻转象限
   - `generatePath()`: 生成完整路径

2. **MortonCode**: 莫顿码算法实现
   - `interleave()`: 位交织编码
   - `deinterleave()`: 位分离解码
   - `generatePath()`: 生成完整路径

3. **CurveVisualizer**: 曲线可视化
   - `draw()`: 绘制曲线
   - `drawGrid()`: 绘制网格
   - `drawPath()`: 绘制路径

4. **App**: 应用控制器
   - 管理用户交互
   - 控制动画播放
   - 更新UI状态
